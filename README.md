<h1>Вычислитель отличий</h1>
<p>Утилита предназначена для выявления различий между двумя структурами данных.</p>
<p>Работает с форматами:</p>
<ul>
<li>.json</li>
<li>.yaml</li>
</ul>
<p>Установка:</p>
<p>$ make install-local</p>
